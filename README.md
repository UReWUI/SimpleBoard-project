# SimpleBoard-project

## 소개
이 프로젝트는 Django를 사용하여 간단한 게시판 기능을 구현했습니다. 회원가입 및 로그인이 된 사용자만이 게시판에 접근가능하며 로그인이 완료된 사용자는 게시글을 작성하고, 수정하고, 삭제할 수 있으며, 자신의 게시글만 수정 및 삭제할 수 있습니다.

## 사용한 프레임워크
- Django 5.1.6

## 기능
- **사용자 인증**
  - 회원가입
  - 로그인
  - 로그아웃
- **게시글 관리**
  - 게시글 작성
  - 게시글 목록 미리보기
  - 게시글 상세 보기
  - 게시글 수정
  - 게시글 삭제
  - 자신의 게시글만 수정 및 삭제 가능
  - 삭제 확인 절차
